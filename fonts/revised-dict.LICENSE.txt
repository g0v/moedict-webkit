revised-dict.woff — MOE Revised-Dict (標楷體2) PUA variant-headword font
=======================================================================

Source: https://dict.revised.moe.edu.tw/revised-dict.woff
Font name table: "(c) 2021 MOE", family "Revised-Dict", version 1.00.
Hosted UNMODIFIED (byte-for-byte; no subset, no WOFF2 re-encode).

Purpose: render MOE's own plane-15 PUA variant headwords (110 codepoints,
U+F0000–FFFFF) used as dictionary titles. Engaged only for those 110
codepoints via a narrowed `unicode-range` piggy-backed on the existing
"Biaodian Pro Serif CNS" family, so no font-family stack is changed and
non-PUA rendering is unaffected. The 9.17 MiB file is fetched only when a
page actually contains one of the 110 PUA codepoints.
